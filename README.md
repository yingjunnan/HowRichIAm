# 我有多富 - 牛马收入计算器 💰

> 🐎 **专为打工人设计的实时收入计算器** - 让你每一秒都知道自己在赚多少钱！

## 🌟 项目特色

### 💸 实时收入显示
- **每秒都在赚钱**：实时显示今日已赚金额，看着数字跳动的感觉太爽了！
- **精确到秒的收入计算**：+¥0.0012/秒，感受时间就是金钱的真谛
- **午休时间提醒**：到点提醒该干饭了，打工人也要劳逸结合嘛～

### ⏰ 智能工作状态追踪
- **上下班倒计时**：距离下班还有 `2时15分30秒`，给个盼头！
- **工作进度条**：今日工作进度 `75%`，成就感满满
- **午休检测**：自动识别午休时间，显示咖啡图标 ☕

### 📊 多维度统计
- **每小时/分钟收入**：让你的时间价值一目了然
- **本月预计收入**：15,000.00 元的小目标
- **已工作天数统计**：这个月已经搬了 `18` 天砖了
- **距离收入目标**：还差 ¥2,500.00 就能达成月度目标！

## 🎨 界面设计

### 🖼️ 动态壁纸系统
- **30+精选壁纸**：从 work1 到 work30，每天都有新感觉
- **自定义壁纸**：支持输入自己的壁纸链接
- **毛玻璃效果**：可调节模糊度，营造梦幻氛围

### 🎨 主题定制
- **6种预设主题色**：蓝、红、绿、紫、橙、青随你切换
- **透明度调节**：0-100% 页面透明度，打造专属视觉效果
- **全屏模式**：一键进入沉浸式赚钱体验

## ⚙️ 强大的配置功能

### 💼 工作参数设置
```
月薪：¥15,000
工作制：5天/8小时（可自定义）
工作时间：09:00 - 18:00
午休时间：12:00 - 13:00
```

### 🔄 灵活刷新频率
- 每0.1秒更新（超实时模式）
- 每1秒更新（标准模式）
- 每5秒更新（节能模式）

### 🧮 智能计算方式
- **按实际自然月天数计算**：2月28天vs3月31天，精准计算
- **排除周末**：周六周日不计算收入，打工人权益保护！

## 🚀 使用方法

1. **打开网页** → 输入你的月薪
2. **设置工作时间** → 上下班时间和午休时间
3. **选择主题壁纸** → 打造你的赚钱氛围
4. **开始享受** → 看着收入数字疯狂增长！

## 💡 特色功能详解

### 🎯 计算公式透明化
点击问号按钮查看详细计算过程：
```
已工作时长: 425.5 分钟
时薪: ¥94.34
今日收入: 425.5 ÷ 60 × ¥94.34 = ¥669.15
```

### 💾 本地数据持久化
所有设置自动保存到本地浏览器，下次打开还是你熟悉的配方！

### 📱 响应式设计
完美适配桌面、平板、手机，随时随地查看你的收入增长。
